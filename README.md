# Dounia Market - E-commerce local pour la diaspora

> Le meilleur des deux mondes : Design clair professionnel + Animations premium

## ✨ Features

- **Design clair** - Fond blanc/crème pour les conversions
- **Animations smooth** - Vue Motion pour les effets
- **Cartes 3D** - Effet de profondeur au hover
- **Typographie premium** - Playfair Display + Inter
- **Mobile-first** - Responsive parfait
- **Trust signals** - Indicateurs de confiance

## 🚀 Installation

```bash
npm install
npm run dev
```

## 🎨 Couleurs

- Primary: #002395 (Bleu Tchad)
- Gold: #D4A853 (Or accent)
- Background: #FAFAF9 (Blanc crème)

## 📦 Stack

- Nuxt 3.14
- Vue Motion
- Tailwind CSS
- Lucide Icons
- Pinia
- Supabase
